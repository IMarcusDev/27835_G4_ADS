## Descripción del Artefacto

La matriz IREB permite la trazabilidad bidireccional entre:
- Requisitos de usuario y requisitos del sistema
- Requisitos funcionales y casos de uso
- Requisitos y componentes de diseño
- Requisitos y casos de prueba
